Initial public release of `@voyantjs/connect-sdk` — the public TypeScript
client for Voyant Connect APIs. Covers operators, connections, links, grants,
oauth-clients, oauth tokens, connector providers and provider registrations,
audit logs, usage, invite tokens, custom connection requests, webhook
subscriptions, the gateway data plane (products, availability, bookings,
suppliers), Connect-normalized inventory reads, and flights (search, price,
book, manage orders, seats, ancillaries, exchange, refund, check-in, SSRs).
